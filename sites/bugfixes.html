Bug Fixes

Pip not found

1. Run the python installation file again
2. Click "Modify"
3. Make sure "pip" is checked
4. Click "Next"
5. Make sure "Add Python to environment variables" is checked
6. Click "Install"

DLL load failed while importing pyarmor_runtime

1. Delete all the python versions that you have installed
2. Download the python version needed for the program (look at the program requirements)

Python was not found

1. Run the python installation file again
2. Click "Modify"
3. Click "Next"
4. Make sure "Add Python to environment variables" is checked
5. Click "Install"
1. Search "Manage App Execution Aliases" in the windows search
2. Disable everything that has "python" in it

AI Aimbot crashing when enabling

1. Update graphics card drivers
2. Download the required cuda toolkit version
3. Change the model from .engine to .pt

Connection closed by remote host

1. Download a VPN (for example https://www.hotspotshield.com)

No matching distribution found for torch==2.1.2

1. Delete all python versions you have installed
2. Download the python version needed for the program (look at the program requirements)
